# Todo List Application

A simple, modular Todo List application built with vanilla JavaScript.

Live Demo :https://sanjeet2610.github.io/Todo-List/

## Features

- Create, edit, and delete todos
- Organize todos within projects
- Mark todos as complete or incomplete
- Assign priority levels to todos
- Set and display due dates in a readable format
- Modal-based form for creating and editing todos
- Persistent data using local storage

## Project Structure

- **Logic layer**  
  Handles todo creation, updates, completion state, and project management.

- **DOM layer**  
  Responsible for rendering todos, projects, and handling user interactions.

- **Storage layer**  
  Saves and restores application state using browser local storage.
